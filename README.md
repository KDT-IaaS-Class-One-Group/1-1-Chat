# Code Talk
## 주제
개발자를 위한 코드 공유에 특화된 채팅 어플리케이션

## 동기
시중에 나와있는 채팅 애플리케이션을 사용하여 개발자 동료들과 코드를 주고 받을 때 복사 붙여넣기 과정에서 각종 오류(유니코드 문제 등), 들여 쓰기 바뀜, 색상 강조 기능 없어짐 등 여러 문제점이 발생함.  
이를 개선하여 개발자에 특화된 채팅 서비스를 직접 구현 하기로 함.
